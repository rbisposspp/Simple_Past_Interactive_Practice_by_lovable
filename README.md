# Simple Past Interactive Practice

Interactive ESL practice for Unit 2 (Simple Past). Everything lives in a single HTML file with embedded CSS and JavaScript.

## Live Preview
- Open `Simple_Past_Interactive_Practice_by_lovable.html` directly in your browser.
- Optional local server for cleaner URLs and asset loading:
  - `python3 -m http.server 8000`
  - Visit `http://localhost:8000/Simple_Past_Interactive_Practice_by_lovable.html`

## Project Structure
```
.
├── Simple_Past_Interactive_Practice_by_lovable.html
├── AGENTS.md
└── README.md
```

## Features
- Multiple activity types (quiz, true/false, cloze, gap-fill).
- Instant feedback, hints, and progress tracking.
- Final session summary with score and motivation message.

## Tech Stack
- HTML + CSS + Vanilla JavaScript (single-file app)
- Google Fonts (Inter)
- `canvas-confetti` via CDN

## Development Notes
- No build step or test runner.
- Keep additions consistent with existing `--esl-*` CSS tokens and `.esl-*` classes.
